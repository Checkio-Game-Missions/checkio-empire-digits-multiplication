Σου δίνεται ένας θετικός ακέραιος αριθμός. Η συνάρτησή σου πρέπει να υπολογίζει το γινόμενο των ψηφίων του αριθμού, εξαιρουμένων των μηδενικών.

Για παράδειγμα: Δίνεται ο αριθμός 123405. Το αποτέλεσμα θα είναι 1*2*3*4*5=120 (μην ξεχάσεις να αποκλείσεις τα μηδενικά).

Είσοδος: Ένας θετικός ακέραιος.

Έξοδος: Το γινόμενο των ψηφίων σε μορφή ακεραίου.

Παράδειγμα: 

checkio(123405) == 120
checkio(999) == 729
checkio(1000) == 1
checkio(1111) == 1

Πώς χρησιμοποιείται: Αυτή η αποστολή θα σου μάθει πώς να λύνεις ένα πρόβλημα που χρησιμοποιεί απλές μετατροπές τύπων δεδομένων.

Προϋποθέσεις: 0 < number < 106

Η σελίδα κάθε αποστολής διατηρεί την ίδια δομή. Μπορείς να βοηθήσεις την κοινότητα να διορθώσει συντακτικά λάθη ή να κάνει πιο ξεκάθαρη την περιγραφή των αποστολών.
